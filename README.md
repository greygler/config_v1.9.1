Конфигуратор данных для LandingPage

1. Состав скрипта:
   Каталог CONFIG - размещается сам конфигуратор
   файл config.php - PHP файл, со сформированными конфигураторами переменными (не обязательно)
        Если файл отсутствует, или Вы случайно\специально его удалили - не беда.
        При первом же обращении к конфигуратору, он будет создан.

2. Установка
   a) Каталог CONFIG и файл config.php разместить в корень сайта, т.е. там, где располагается Index.php
   b) В самом верху файла index.php и файла со скриптом обработки форм разместить строку <? include("config.php") ?>
      !!! Если Ваш INDEX имеет расширение HTML (то есть называется index.html) его нужно переименовать в index.php
   c) Заменить все соответствующие данные на переменные таким образом <?= ИМЯ_ПЕРЕМЕННОЙ ?>
   
   Например, все новые цены заменить на <?= $price_new ?>

3. Использование.
   а) Вызываем http://ИМЯ_САЙТА/config
   
   Стартовый Логин:  admin
   Стартовый Пароль: admin
   
   b) Для начала выбираем функцию "Сменить пароль" и меняем логин и пароль на свои
   с) Авторизуемся с новывым логином и паролем, выбираем функцию "Конфигурация" и задаем свои настройки.
   
   d) Сохраняемcя, и на забываем выйти.
   
4. Расширение возможностей. Для тех, кто знает HTML хотя бы на начальном уровне!

   Системой предусмотренно расширение возможных переменных.
   Для этого 
   a) любым редактором откройте файл config.php, который находится в папке CONFIG (не перепутайте! Именно тот, что в папке!)
   b) Найдите закомментированную строку 
   
    <!--	 Дополнительная переменная в разделе НАЗВАНИЕ РАЗДЕЛА -->
	
   с) Отредактируйте (вставьте в нужное место имя перемнной и метку_ID) и ее вставьте вместо коммента.
   
   Если нужна строка:
   
   <label for="МЕТКА_ID1">НАЗВАНИЕ ПОЛЯ<em>*</em></label><input required id="МЕТКА_ID1" type="text" name="ИМЯ_ПЕРЕМЕННОЙ1" value="<?=  $ИМЯ_ПЕРЕМЕННОЙ1 ?>" placeholder="ТЕКСТ, КОТОРЫЙ БУДЕТ НАПИСАН ВНУТРИ ПОЛЯ"></p>оле:
   
   Если нужно текстовое поле:
   
   <label for="МЕТКА_ID2">НАЗВАНИЕ ПОЛЯ: <span><br>Пояснение к полю.</span></label><textarea rows="5" id="МЕТКА_ID2" name="ИМЯ_ПЕРЕМЕННОЙ2" cols="70"><?= str_replace("<br />", "\n", $ИМЯ_ПЕРЕМЕННОЙ2); ?></textarea></p>
